Overview

The protocol that describes EMFaceAuthEnrollerDelegate methods

– enroller:didUpdateFaceAuthFrameEvent: required method

The delegate method invoked during frame updates. The frameEvent contains the information of the frame captured by the SDK. Application should extract the information and display the image to the user as well as prompting the user about the status of the enrollment. e.g. ‘blinking’ is requried to complete the enrollment. See EMFaceAuthLivenessAction.

- (void)enroller:(id<EMFaceAuthEnroller>)enroller didUpdateFaceAuthFrameEvent:(id<EMFaceAuthFrameEvent>)frameEvent

Parameters

enroller

The enroller object of the method.
frameEvent

The frame event. See EMFaceAuthFrameEvent.
