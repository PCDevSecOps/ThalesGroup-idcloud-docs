  matchingThreshold

Confidence of the matching. The value must be in the range [0,72]. The higher the value is set, the lower the false acceptence rate (FAR) will be.

Default value is 48.

  livenessBlinkTimeout

Face liveness blink timeout in milliseconds. The value must be in the range [0,30000]. The face verifier will return an error if no eye blink is detected within the timeout during face verification.

Default value is 5000.
