EMFaceAuthSettings Class Reference

Inherits from NSObject
Declared in EMFaceAuthSettings.h

Overview

It defines common face authentication settings to be used in enrollment and verification.

  faceCaptureMode

The capture mode to use. Image mode should be used to enroll a user, liveness mode is highly recommended for user verification.

Default mode for enrollment is EMFaceCaptureModeImage. Default mode for verification is EMFaceCaptureModeLivenessPassive.

@property EMFaceCaptureMode faceCaptureMode

  livenessThreshold

The liveness threshold. This threshold is used in liveness mode only and determine how long the ‘still phase’ will be and how strict the check of liveness will be. The value must be in the range [0, 100].

Default value is 0, so no ‘still phase’ is required.

@property uint8_t livenessThreshold

  qualityThreshold

Quality of the face detection to be achieve. This value is used to reject bad quality images. The value must be in the range [0,100].

Default value is 48.
