Overview

The delegate methods during license management.

– licenseBuilderShouldFetchLicenseFromServer:

The method to implement whether or not to dispatch request to acquire license from server. If license file is not found or has become invalid. If YES is returned, will proceed the acquisition of license, otherwise aborts the process Default is YES if not implemented

- (BOOL)licenseBuilderShouldFetchLicenseFromServer:(EMFaceAuthLicenseBuilder *)licenseBuilder

Parameters

licenseBuilder

The license builder.

Return Value

YES to proceed the process or NO to abort
