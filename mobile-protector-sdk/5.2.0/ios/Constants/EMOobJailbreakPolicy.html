EMOobJailbreakPolicy

The OOB behavior policy of the Ezio Mobile SDK when the physical device is detected as being rooted.

Definition

typedef NS_ENUM(NSInteger, EMOobJailbreakPolicy ) {
   EMOobJailbreakPolicyIgnore,
   EMOobJailbreakPolicyFail,
   EMOobJailbreakPolicyUnregister,
};

Constants

EMOobJailbreakPolicyIgnore

No action is taken when calling any OOB related API of Ezio Mobile SDK even the physical device is jailbroken.

Declared In EMOobJailbreakPolicy.h.

EMOobJailbreakPolicyFail

A RootPolicyException is thrown when calling a method in the Ezio Mobile SDK that detects if the physical device is jailbroken. Notice! This is the default jailbroken policy.

Declared In EMOobJailbreakPolicy.h.

EMOobJailbreakPolicyUnregister

A RootPolicyException is thrown and client is silently unregistered when calling a method in the Ezio Mobile SDK that detects if the physical device is jailbroken.

Declared In EMOobJailbreakPolicy.h.
