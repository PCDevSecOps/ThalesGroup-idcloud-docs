EMFaceAuthVerifier Protocol Reference

Conforms to NSObject
Declared in EMFaceAuthVerifier.h

Overview

EMFaceAuthVerifier provides functions to verify or authenticate a user.

– authenticateUser:withTimeOut:completionHandler: required method

Authenticate user by face ID. The application could customize the action upon error cases. The completion handler will return an EMFaceAuthInput object upon success that can be used to generate OTPs.

- (void)authenticateUser:(id<EMAuthenticatable>)authenticatable withTimeOut:(int)timeout completionHandler:(void ( ^ ) ( id<EMFaceAuthInput> faceAuthInput , NSError *error ))completionHandler

Parameters

authenticatable

the target authenticatable item (e.g. CapToken)
timeout

the timeout for verification process
completionHandler

completion handler to be called with the result once it’s completed.
