IDCTransactionContext

@interface IDCTransactionContext : NSObject

An interface that describes the transaction to be signed.

The developer should show the transaction information to the application user.

@since 1.0

  • content

    Returns the dictionary containing transaction details.

    @since 1.0

    After time elapses, if there is no action triggered, operation will fail.
