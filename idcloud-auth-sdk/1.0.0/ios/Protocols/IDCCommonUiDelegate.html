IDCCommonUiDelegate

@protocol IDCCommonUiDelegate <NSObject>

UI Delegate that contains methods for managing authenticator selection and transaction confirmation.

@since 1.0

  • -idCloudClient:authenticators:userChoiceHandler:cancelHandler:

    Called during authenticator selection.

    The developer should show authenticators to the user and let user choose which to use.

    This method will not be called if one of following condition is fulfilled:

    • There is only one authenticator option in the request.

    • There are two authenticators option; but system biometric is not enrolled. In this case, PIN authenticator will be selected.

    @since 1.0

  • -idCloudClient:confirmTransaction:confirmHandler:cancelHandler:

    Called during transaction confirmation.

    The developer should show transactionContext to the user and let user decide whether to accept the transaction.

    Transaction confirmation has timeout on which it waits for handler to be called. The timeout value is obtained by IDCTransactionContext.timeout. If the time elapses without any calls to the handler, the operation will fail and any further call to the handlers will be ignored.
