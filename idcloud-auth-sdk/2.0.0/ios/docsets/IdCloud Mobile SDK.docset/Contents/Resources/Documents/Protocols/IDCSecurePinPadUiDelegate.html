IDCSecurePinPadUiDelegate

@protocol IDCSecurePinPadUiDelegate <NSObject>

An UI delegate that contains methods for managing PIN authentication.

@since 1.0.0

  • -idCloudClient:willEnterPin:scenarioContext:cancelHandler:

    Called when a PIN authentication starts.

    The developer should show securePinPad to the user.
