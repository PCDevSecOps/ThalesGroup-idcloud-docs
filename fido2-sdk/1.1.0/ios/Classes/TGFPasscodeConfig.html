TGFPasscodeConfig

@interface TGFPasscodeConfig : NSObject

A global Passcode configuration class. In general FIDO2 SDK’s default behavior is to choose reasonable and secure defaults. An application may however wish to deviate from the default behavior to make Passcode verification easier for the user.
