TGFAuthenticator

@protocol TGFAuthenticator <NSObject>

Abstract class to describe the features provided by each of its subclassed authenticators.

@since 1.0.0

  • -createPasscode

    Create a Authenticator. Users MUST be enrolled to the specific authenticator to perform FIDO2 operations. The enrollment process stores a secure means of identification which will be used to verify the user when authentication is required.

  • -deletePasscode

    Unenroll a previously enrolled Authenticator. User authentication is required prior to a successful unenrollment. Upon successful unenrollment, all FIDO2 registrations for the specific authenticator will be invalidated. A subsequent enrollment is required to once again perform FIDO2 operations.
