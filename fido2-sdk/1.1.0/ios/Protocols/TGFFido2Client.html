TGFFido2Client

@protocol TGFFido2Client <NSObject>

A FIDO2 client that processes FIDO2 requests and generates FIDO2 responses.

@since 1.0.0

  • -respondWithArgs:completion:

    Responds asynchronously to a FIDO2 request by generating a FIDO2 response. This operation may run for a long period of time and all UI is delegated to the caller.

    @since 1.0.0

    Declaration

    Objective-C

                                      NSError *_Nullable))completionBlock;

    Swift

    func respond(with respondArgs: TGFFido2RespondArgs, completion completionBlock: @escaping (TGFFido2Response?, Error?) -> Void) -> TGFFido2Task?

    Parameters

    respondArgs

    FIDO2 request plus other arguments required to fulfill the request.
    completionBlock

    The completion handler to call when the execution is complete. It takes the following parameters:

    • response - Success when it is not nil.
    • error - Error details. See TGFFido2Error for error types.
