TGFPasscodeAuthenticatorDelegate

@protocol TGFPasscodeAuthenticatorDelegate <NSObject>

A delegate to show the keyboard view and handle the Passcode enrollment / unenrollment events. Callbacks from this delegate will inform the App layer of the outcome of these processes as well as to display the secure keyboard UI.

Note

All callbacks are invoked on the Main thread.

@since 1.0.0
