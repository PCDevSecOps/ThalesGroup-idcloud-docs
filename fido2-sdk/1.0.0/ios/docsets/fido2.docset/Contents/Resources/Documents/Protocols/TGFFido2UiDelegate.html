TGFFido2UiDelegate

@protocol TGFFido2UiDelegate <NSObject>

A delegate class when FIDO2 SDK must interact with the user because FIDO2 SDK does not display UI and relies on the application to interact with the user.

Note

All delegate callbacks are invoked on the Main thread.

@since 1.0.0
