TGFKeyboard

@protocol TGFKeyboard <NSObject>

A class to construct and return a secure keyboard as a UIView object and to subscribe to relevant keyboard events.

@since 1.0.0

  • -view

    A secure keyboard to be presented for user verification. This UIView object should be displayed on the App layer to provide a secure passcode entry.

    Note

    Multiple calls to this method would return the SAME object instance. A new keyboard view is only generated on the first call of the object.

    Declaration

    Objective-C

    - (UIView *_Nonnull)view;

    Swift

    Return Value

    a UIVew of the secure keyboard
